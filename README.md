# Piktura - Wallpaper App

Aplicativo para explorar, baixar e definir papéis de parede de alta qualidade.

## Características

- Grid de papéis de parede
- Visualização em tela cheia
- Download de wallpapers
- Definir como papel de parede nativo
- Editor de wallpapers personalizado
- Mais de 20 fontes disponíveis
- Integração com Firebase e AppLovin

## Requisitos

- Android 5.0 (API 21) ou superior
